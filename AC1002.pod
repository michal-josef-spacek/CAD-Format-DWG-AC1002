=head1 NAME

CAD::Format::DWG::AC1002 - Read AutoCAD r2.5 DWG file.

=head1 SYNOPSIS

 use CAD::Format::DWG::AC1002;

 my $dwg = CAD::Format::DWG::AC1002->new;
 my $dwg = CAD::Format::DWG::AC1002::from_file($dwg_file);
 my $header_hr = $dwg->header;
 my $entities_hr = $dwg->entities;

=head1 DESCRIPTION

In distribution is kaitai structure definition file dwg_ac1002.ksy which
described AutoCAD DWG format version AC1002.

=head1 DEPENDENCIES

L<IO::KaitaiStruct>.
